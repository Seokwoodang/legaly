# 01 · 작업 스펙 v2 (델타) — 리걸리 프론트엔드

> v1(`../v1/01-working-spec.md`) 대비 **델타**. 원본은 `../source/v2-design_files/` 보관.
> v1에서 유지되는 동작은 그대로 두고, 아래 변경/추가/삭제만 적용한다.

## 스펙 diff 분류 (U2)

### 🆕 추가(Added)
1. **절차 가이드 기능** — 신규 2페이지 + 데이터.
   - `/guides` 목록: 팀별 가이드 카드(분야 틴트 아이콘, 분야 칩, 검색), "총 N개".
   - `/guides/:id` 상세: 히어로(분야 틴트, 기간·난이도·단계) + 사이드바(필요서류·기한·주의·비용) +
     단계별 절차(번호·연결선·팁) + 근거 법령(→/laws/:id) + 참고 판례(→/cases/:id) + CTA(→상담).
   - 데이터: 가이드 10개(법무 7·세무 3). 각 가이드 = {id, team, field, title, summary, duration,
     difficulty, intro, steps[{t,d,tip?}], documents[], deadlines[{k,v}], costs?[{k,v}], laws[{title,article,note}], cases?[{name,number}]}.
2. **분야→아이콘 틴트 시스템** — `FIELD_TINT`(분야→{bg,fg,svg}) + `ICON`(SVG 세트) + `FIELDIC`
   (목록 분야→아이콘키). 분야 배지·카드·가이드 히어로에 사용.
3. **서브내비 "절차 가이드" 탭** — 모든 팀 페이지: 상담·**절차 가이드**·자료실·판례·보관함
   (서브내비 `overflow-x:auto`, 5탭).
4. **로비 신규 섹션:** ① 질문 박스(팀 선택 토글 + 입력 + "물어보기 →" + 예시 칩), ② 분야별 둘러보기
   (8개 분야 카드 → /guides 해당 분야), ③ 자주 찾는 절차 가이드(6개 featured → /guides/:id).

### ✏️ 수정(Modified)
1. **아이콘 시스템 교체(횡단):** 한자 글리프(法/稅/民/刑/商/勞/行/家/不) → **SVG 아이콘**.
   - 로고=scale, 법무 세그먼트/아바타=scale, 세무=calc. 자료실/판례 행 분야 배지: `FIELDIC[field]`→ICON.
   - ai 상담 AI 아바타: 팀 아이콘(scale/calc). 자료실/판례 히어로 팀 배지 아이콘.
   - 인증 모달 로고도 SVG(로비). (그 외 페이지 모달 로고는 디자인상 한자 잔존 — §그리드 참고)
2. **로비 헤더:** 팀 세그먼트/서브내비 없음 유지하되 상단 링크 추가(절차 가이드 / AI 상담) + 계정.
   로그인 드롭다운에 이름 헤더 행 추가.
3. **로비 히어로 카피:** 비로그인 "법이 필요한 순간,\n혼자 막막하지 않게." / 로그인 "○○님, 무엇을 도와드릴까요?".

### 🗑️ 삭제(Removed) — 로비에서 제거
1. **팀 카드 2개**(나의 법무팀/세무팀 + 상담 대기 중 + 예시칩 + CTA) — 질문 박스+분야+가이드로 대체.
2. **보관함 미리보기 섹션**(로비의 "내 보관함" 카드/빈상태). 보관함은 서브내비/계정 드롭다운으로 접근 유지.
   → v1 로비 테스트(카드 제목·미리보기·예시칩 pendingQ) 폐기/교체.
