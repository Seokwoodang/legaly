# 08 · 완료 보고 v2 (Gate 2) — 리걸리 프론트엔드

## 이번 업데이트(v2)가 한 일
v1(법률·세무 AI 상담 + 자료실/판례/보관함) 위에 **절차 가이드 + 로비 개편 + 아이콘 시스템**을 추가.

- **절차 가이드 신설**
  - `/guides` 목록 — 팀별 가이드 카드(분야 틴트 아이콘·분야 칩·검색), `?field=` 딥링크.
  - `/guides/:id` 상세 — 단계별 절차(번호·연결선·팁), 필요 서류·기한·비용 사이드바, 근거 법령(→자료실)·
    참고 판례(→판례), "내 상황엔?" CTA(상담 자동전송), 미상 id 폴백.
  - 가이드 10개(법무 7·세무 3): 고소·고발, 부당해고, 임금체불, 전세보증금, 지급명령, 상속포기, 협의이혼,
    종합소득세, 양도세 비과세, 상속세.
- **로비 개편** — 팀 카드·보관함 미리보기 제거 → ① 질문 박스(팀 선택+입력+예시→상담), ② 분야별 둘러보기
  (8개→가이드), ③ 자주 찾는 가이드(6개). 헤더 상단 링크(절차 가이드/AI 상담).
- **아이콘 시스템** — 한자 글리프(法/稅/民…) → SVG 라인 아이콘(JSX 정적 렌더). 로고·팀 세그먼트·상담
  아바타·자료실/판례 행/히어로·가이드 분야 틴트에 적용.
- **서브내비** — 상담·**절차 가이드**·자료실·판례·보관함(가로 스크롤).

## 핵심 설계
- 가이드 데이터/매핑은 `src/lib/guidesData.ts`(순수), 아이콘은 `src/components/Icon.tsx`(JSX 정적).
- 데이터 접근은 v1과 동일하게 `dataClient`(getGuides/getGuide 추가)·라우팅 파라미터(`/guides/:id`,
  `?field=`). 영속 키 신설 없이 URL로 딥링크.
